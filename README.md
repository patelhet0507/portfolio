# Het Patel — Portfolio

Personal portfolio built as a single-page static site. Dark theme, particle background, interactive terminal, and scroll-triggered animations.

## Sections

- **Home** — Hero with gradient title, badge, social links
- **About** — Bento grid with bio, education, highlights, focus areas, stats counters
- **Experience** — Timeline with glowing dots and rail gradient
- **Skills** — Animated SVG radial progress rings with tech icons
- **Projects** — Project cards with mouse-reactive glare overlays and live demo links
- **Terminal** — Interactive CLI emulator with resume/projects/skills/about/help/clear commands

## Tech

- Vanilla HTML/CSS/JS — no frameworks, no build tools, no dependencies
- Google Fonts (Inter + JetBrains Mono)
- Particle canvas background
- IntersectionObserver for scroll animations
- Responsive down to mobile
